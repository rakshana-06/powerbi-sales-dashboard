# Power BI Sales Dashboard

## Project Overview
This project presents an interactive dashboard built using Power BI to analyse retail sales data.

## Tools Used
- Power BI
- Excel

## Dataset
The dataset includes:
- Order ID
- Order Date
- Product
- Category
- Quantity
- Sales
- Region

## Dashboard Features
- Sales overview (Total Sales, Quantity)
- Category-wise analysis
- Product-wise performance
- Region-wise distribution
- Sales trend over time

## Data Cleaning
- Removed missing values
- Corrected data types
- Cleaned column names
- Prepared data using Power Query

## Conclusion
The dashboard provides a clear and visual representation of sales data, helping to understand patterns and trends.
